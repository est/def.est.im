inspire
/ɪnˈspaɪər/
词源：来自拉丁语 inspirare（in- ‘进入’ + spirare ‘呼吸’），原义“注入精神、吹入灵气”，后引申为“启发、鼓舞”。
字面意思：将某种(思想/情感)“吹入”或“注入”。
来源: 拉丁语
频率: 常用
语域: 中性/正式
