inspire

/ɪnˈspaɪər/
verb

Etymology

from Latin inspirare “to breathe into”; later via Old French inspirer. (原义：向内呼吸 / 注入气息)

First attested

late Middle English (c. 14th–15th cent.)

Literal breakdown

in- (into) + spirare (to breathe) → “breathe into”
